# AURA Boxing

Premium boxing/lifestyle brand landing page project for the AURA launch experience.

## Stack

- Vite
- React
- CSS modules by section in `src/styles`
- Central route config in `src/data/routes.js`

## Routes

- `/`
- `/shop`
- `/collections/drop-001`
- `/campaign`
- `/about`
- `/contact`
- `/size-guide`
- `/shipping`
- `/returns`
- `/privacy-policy`
- `/terms`
- `/join`
- `/account`
- `/cart`
- `/products/coming-soon`

## Development

```bash
npm install
npm run dev
```

## Notes

The project mirrors the current AURA Shopify draft experience and uses the Shopify-hosted launch imagery referenced in `src/data/assets.js`. Public SVG and metadata assets live in `public/assets`.
